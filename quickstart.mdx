---
title: Quickstart
description: "Get your first trace into bryel in a couple of minutes."
---

<Steps>
  <Step title="Get an API key">
    Create a project in the [bryel dashboard](https://bryel.ai) and copy its API
    key (`bk_…`). Set it as `BRYEL_KEY`.
  </Step>
  <Step title="Install the SDK">
    Pick the package for your stack:

    <CodeGroup>
    ```bash Vercel AI SDK
    npm i @bryel/vercel @vercel/otel
    ```
    ```bash Any OTel app
    npm i @bryel/sdk-core @opentelemetry/sdk-trace-base
    ```
    </CodeGroup>
  </Step>
  <Step title="Register the processor">
    For the **Vercel AI SDK** (`instrumentation.ts`):

    ```ts
    import { registerOTel } from "@vercel/otel";
    import { bryelVercelProcessor } from "@bryel/vercel";

    export function register() {
      registerOTel({
        serviceName: "my-app",
        spanProcessors: [bryelVercelProcessor({ apiKey: process.env.BRYEL_KEY! })],
      });
    }
    ```
  </Step>
  <Step title="Enable telemetry per call">
    ```ts
    const result = streamText({
      model: openai("gpt-4o"),
      prompt,
      experimental_telemetry: {
        isEnabled: true,
        metadata: { sessionId, userId }, // groups a conversation in bryel
      },
    });
    ```
  </Step>
  <Step title="See it in bryel">
    Run your app, then open the dashboard — you'll see the run: streamed turns,
    tool calls, tokens, and cost.

    <Tip>A streamed span finalizes only once the stream is fully consumed — always drain it.</Tip>
  </Step>
</Steps>

Not on Vercel? See [Any OpenTelemetry app](/frameworks/opentelemetry).
